# CRM Influencer Management Dashboard

基于达人 CRM 信息重构的静态交互看板，用于管理达人销售表现、跟进流水线、商务分组、个人贡献和分析报表。

## Preview

直接打开 `index.html` 即可预览，也可以通过任意静态服务器托管。

## Features

- 跟进阶段：待建联、建联中、已寄样、待排期、已合作、深度合作。
- 筛选维度：产品、组别、达人玩法、达人类型、达人等级、状态、商务、关键词。
- 数据看板：核心 KPI、销售额拆分、达人状态占比、个人销售额排行、达人订单、商务跟进量。
- 记录管理：查看达人详情、编辑达人信息、推进/退回阶段、维护卡点备注。
- 指标编辑：可维护当月销售额、上月销售额、全年目标和全年已完成销售额。
- 导出：按当前筛选条件导出 CSV。
