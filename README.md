# CRM Influencer Management Dashboard

一个基于 `达人CRM看板.html` 信息重构的静态交互原型，用于管理达人跟进流水线、筛选达人池、查看卡点、分析类型/组别/玩法分布，并维护达人记录。

## 预览

直接打开 `index.html` 即可预览，也可以通过任意静态服务器托管。

## 功能

- 跟进阶段：待触达、已触达、沟通中、已寄样、试播中、洽谈排期、已签约、已流失。
- 筛选维度：产品、组别、达人玩法、达人类型、达人等级、状态、商务、关键词。
- 数据看板：核心 KPI、流水线漏斗、达人类型分布、各组活跃达人、玩法结构。
- 记录管理：添加达人、编辑达人、推进/退回阶段、查看详情。
- 导出：按当前筛选条件导出 CSV。
